   - Behavioral guidelines

   - Component structure
   - Integration points
   - Visual diagram

   - Available modes
   - Activation commands

   - Evaluation criteria

   - Success thresholds

   - Required documents
   - Reference materials
   - Supporting resources

   - Phase breakdown
   - Step-by-step process
   - Testing approach
